# Ficruty Project Roadmap

This document outlines the strategic milestones for Ficruty (caocap). Our goal is to move from a spatial foundation to a full-fledged agentic development environment.

## Phase 1: Foundations (Current)
*Focus: Establishing the core spatial runtime and infrastructure.*

- [x] **Spatial Engine**: Infinite grid with gesture-driven pan/zoom.
- [x] **Node Infrastructure**: Customizable spatial nodes with persistence.
- [x] **Architecture Refinement**: Transition to `@Observable` and domain-driven folder structure.
- [x] **Monetization (Pro)**: Initial StoreKit 2 integration for premium tiers.
- [ ] **Node Linking**: Visual Bezier-curve connections between arbitrary nodes.
- [ ] **Omnibox Search**: Search-to-fly functionality to navigate large canvases.

## Phase 2: Agentic Intelligence
*Focus: Integrating AI deeply into the spatial workflow.*

- [ ] **CoCaptain Integration**: Fully functional AI sidekick for code generation and intent refinement.
- [ ] **Context Awareness**: Teaching the agent to "read" the entire spatial canvas to provide better suggestions.
- [ ] **Intent-to-Node**: Transforming natural language descriptions into functional code nodes automatically.

## Phase 3: The Code Runtime
*Focus: Making the nodes "alive."*

- [ ] **Live Previews**: Integrating high-performance WebViews to see code execution in real-time.
- [ ] **Spatial Debugging**: Visualizing variable flow and execution state across the canvas.
- [ ] **File System Bridging**: Bi-directional sync between the spatial canvas and traditional Git repositories.

## Phase 4: Collaborative Ecosystem
*Focus: Bringing developers together.*

- [ ] **Real-time Collaboration**: Multi-user spatial canvases with shared agentic history.
- [ ] **Mobile-First Experience**: Optimizing complex code interactions for iPad and iPhone touch interfaces.
- [ ] **Plugin System**: Allowing third-party developers to create custom node types and agent behaviors.

---

> [!NOTE]
> This roadmap is a living document. As we "vibe code" and discover new possibilities, these milestones will evolve.
